<h1 align="center">📌 TIMELINE REACT</h1>

<p align="center">
  Uma timeline interativa construída com React, inspirada em redes sociais modernas.<br>
  O projeto busca <b>simplicidade</b>, <b>performance</b> e <b>organização</b> no desenvolvimento front-end.
</p>

</p>

<hr>

<h2>✨ TECNOLOGIAS UTILIZADAS</h2>
<ul>
  <li>⚛️ <b>React</b> → Biblioteca JavaScript para criar interfaces de usuário dinâmicas e reativas.</li>
  <li>⚡ <b>Vite</b> → Ferramenta de build rápida com Hot Module Replacement (HMR).</li>
  <li>💅 <b>CSS Modules</b> → Estilização modular que evita conflitos e facilita manutenção.</li>
</ul>
