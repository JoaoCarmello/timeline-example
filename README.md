# Timeline React

Uma timeline interativa construída com React, inspirada em redes sociais.

## ✨ Tecnologias Utilizadas

*   ⚛️ **React:** Biblioteca JavaScript para criar interfaces de usuário dinâmicas e reativas.
*   ⚡ **Vite:** Ferramenta de build que oferece um desenvolvimento extremamente rápido com Hot Module Replacement (HMR).
*   💅 **CSS Modules:** Estilização modular que encapsula o CSS dentro de componentes, evitando conflitos e facilitando a manutenção.
